Lab 1: Initial Setup

First, we start with most basic blink the led excersice. This is a very beginner experiment similar to the 'hello world' type of starter as show below gif.

Second, we used a example code to verify that the IMU on the development board could work properly. Where IMU is an inertia measurement unit that measure acceleration and rotation of object.

Third, we modify the example code a bit to verify that the oled display can work.

Fourth, we also tried with storing some sensor data into the SD card.

fifth, we also ultize the RTC (real time clock) to print out the current time of our computer.

finally, we sound the buzzer to a traditional 'happy birthday' song.
